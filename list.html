If you want to have a list of items where the order doesnâ€™t matter, like a shopping list of items that can be bought in any order, then you can use an unordered list. Unordered lists are created using the "ul" element, and each item within the list is created using the list item element "li".

Each list item in an unordered list begins with a bullet point

Example

  • Item 1
  • Item 2
  • Item 3

If you instead want to create a list of items where the order does matter, like step-by-step instructions for a recipe, or your top 10 favorite TV shows, then you can use an ordered list. Ordered lists are created using the "ol" element. Each individual item in them is again created using the list item element "li". However, each list item in an ordered list begins with a number instead:

  1. Item 1
  2. Item 2
  3. Item 3

Ordered lists also have a unique attributes. These include "start" and "reversed".

While many ordered lists start with the number 1, the "start" attrubute allows you to start the list at any number you want. in the following example we will start from the number 23

  1. This item will start at the number 23
  2. 24
  3. And finally, the number 25

For the attrubute "reversed" the numbers will be in decending order insted of acending

  1. Countdown, 3
  2. 2
  3. 1

Another attribute is "value". This will change the number associated with the list.

  1. Item 1
  2. Item 4
  3. Item 5
  4. Item 6
  5. Item 10
  6. Item 11

In that example, I changed the value of the second item to the number 4, and the value of the fifth item to the number 10.

Description lists are also seen, but not as often as ordered and unordered lists. For discription lists you will use the "dl" instead of "li". The discription list requires two elements to function correctly, "dt", which is the team element, and "dd" which is the description element.
